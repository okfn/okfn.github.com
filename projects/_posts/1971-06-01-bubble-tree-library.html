--- layout: project title: Bubble Tree Library projecturl: http://okfnlabs.org/bubbletree/ imageurl: http://driven-by-data.net/img/gallery/bubbletree-1.png type: Library tags: author: Gregor Aisch & David McCandless authorurl: http://driven-by-data.net/ github_user: okfn github_repo: bubbletree permalink: /projects/bubble-tree-library/index.html slug: bubble-tree-library --- The BubbleTree can be used to display hierarchical (spending) data in an interactive visualization. The setup is easy and independent from the OpenSpending platform. However, there is an optional integration module to connect with data from the OpenSpending API.
