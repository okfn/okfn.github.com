--- layout: project title: ReclineJS - Build Data Applications in the Browser projecturl: http://okfnlabs.org/recline/ imageurl: http://farm8.staticflickr.com/7251/7508402742_03c74a9763_c.jpg type: Library tags: author: Rufus Pollock, Max Ogden & others authorurl: https://github.com/okfn/recline#contributors github_user: okfn github_repo: recline permalink: /projects/reclinejs/index.html slug: reclinejs --- Recline is a simple but powerful library for building data applications in pure Javascript and HTML. Building on Backbone, Recline supplies components and structure to data-heavy applications by providing a set of models (Dataset, Record/Row, Field) and views (Grid, Map, Graph etc).
